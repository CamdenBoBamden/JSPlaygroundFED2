Intro to JS

Arrays

The color displayed above was selected for you from an array.

Functions

Functions can be written in JS and called upon in reaction to an event. Allow a user to enter a color. When they click the button (event), call a function that will display a message and user input.

Water Weight

Assume a gallon water weighs 8.33lbs. Allow a user to enter a number of gallons. Upon clicking a button, call a function to calculate the total weight and display that value.

Loops

Looping also exits in JS. Use a for loop to display 1-10.

Single Milestone

Allow a user to enter their age. When they click a button, call a function that uses a switch to determine if there any milestones for them

Multiple Milestones

Allow a user to enter their age. When they click a button, call a function that use if/else to display all milestones they have reached.
